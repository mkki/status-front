> 본 문서는 기획 문서에서 선택 화면 UI와 소셜 로그인 종류만 명시되어 있고, 세부 플로우(2. 로그인, 3. 회원가입, 4. 홈 화면)는 비어있는 상태. 추후 기획 보강 예정.

## 1. 로그인/회원가입 선택 화면

앱 실행 시 출력되는 로딩 화면 이후 본 로그인/회원가입 선택 화면을 출력한다.

단, 자동 로그인이 되어있는 상태라면 본 화면을 생략하고 홈 화면으로 이동한다.

- 자동 로그인은 아래 2. 로그인 목차에서 설명한다.
- 홈 화면은 아래 4. 홈 화면 목차에서 설명한다.

로그인 및 회원가입 방법으로 소셜 로그인만 지원한다.

지원하는 소셜 로그인은 다음과 같다.

- Apple ID 로그인
- Google ID 로그인
- 카카오톡 ID 로그인

소셜 로그인 구현 시 해당 업체의 기본 API를 따른다.

(목업 수정 제안_디자인 쪽에서 결정합니다.)

로그인/회원가입 페이지에는 다음을 출력한다.

**1) 앱 이름**

Text: 상태창

**2) 앱 부제**

Text: 현실 판타지 자기계발 앱

**3) 앱 아이콘**

상태창 앱 아이콘을 출력한다.

- 리소스는 Figma를 참고한다.
