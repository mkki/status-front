# 능력치 시스템

## 1. 개요

능력치는 유저가 챌린지를 수행하거나, 목표를 달성했을 때 규칙에 의해 성장한다.

즉, 능력치는 사용자의 현실 행동을 게임화하여, 정신적·기술적 역량의 성장 과정을 수치화한 시각적 지표로써 성장/성취를 체감할 수 있는 핵심 정보로 활용된다.

추후 플레이어블 콘텐츠 구현 시 능력치가 플레이 조건으로 활용된다.

능력치는 '정신 능력치'와 '기술 능력치'로 구분한다.

## 2. 정신 능력치

### 2.1 정의

정신 능력치는 사용자의 **내면적 태도, 습관, 사고방식 등 비가시적인 역량**을 나타낸다.

### 2.2 항목

| 능력치 | 설명 | 매핑 포인트 | 해당 퀘스트 유형 예시 |
| --- | --- | --- | --- |
| **인내** | 실행 저항을 이겨내고 끝까지 밀어붙이는 내적 추진력 | 고난도 실행, 반복 회피 극복, 심리적 저항 돌파 | 기상 루틴, 체력 단련 루틴, 습관 재개, 불쾌한 일 미루지 않기 등 |
| **집중** | 몰입 상태를 유지하고 방해 요소를 차단하는 정신력 | 주의 통제, 타이머 활용, 외부 자극 차단 | 타이머 집중 루틴, SNS 차단, 집중 시간 확보, 특정 장소에서 작업 등 |
| **제어** | 감정과 욕구를 인식하고 절제하는 자기조절 능력 | 충동 억제, 감정 인지, 선택적 행동 | 감정 기록, 지출 관리, 야식/음주 제한, 선택적 행동 실천 등 |
| **영감** | 새롭고 창의적인 아이디어를 떠올리고 구상하는 상상력 | 아이디어 생성, 자유 구상, 창작 시도 | 브레인스토밍, 콘셉트 스케치, 자유 글쓰기, 창작 노트 작성 등 |
| **성실** | 단조로운 반복을 기꺼이 수행하는 루틴 기반의 꾸준함 | 반복 훈련, 체크리스트, 정기 수행 | 루틴 실천, 매일 일기, 체크리스트 기록, 주간 반복 과제 이행 등 |
| **용기** | 두려움과 불확실성을 넘어서 행동하는 실천 기반 도전성 | 낯선 환경 도전, 미지 탐색, 대외 활동 시도 | 모임 참석, 발표 신청, 낯선 장소 방문, 새로운 행동 시도 등 |

## 📌 매핑 시 참고 기준

- **내적 저항/심리적 마찰**을 요소로 삼는 퀘스트라면 → **의지력**
- **감각 자극/주의 분산 차단**이 핵심이면 → **집중력**
- **감정/욕구를 다루거나 억제**하는 요소가 있으면 → **자기 통제력**
- **새롭게 구성하거나 창조**하는 사고가 필요하면 → **창의성**
- **지루한 루틴의 반복**을 기반으로 하면 → **성실성**
- **불안, 낯섦, 미지의 행동**이 핵심이면 → **대담성**

## 3. 기술 능력치

### 3.1 정의

기술 능력치는 자기계발, 커리어, 창작, 건강관리, 일상 실행력 등 **직접 활용 가능한 실천 기술 영역**을 표현한다.

### 3.2 항목

| --- | --- | --- | --- |
| **건강** | 신체적 수련과 생리적 루틴 관리 능력 | 신체 활동, 컨디션 유지, 리듬 조절 | 운동 루틴, 스트레칭, 수면 관리, 식사 습관 개선, 명상, 휴식 등 |

| **탐구** | 새로운 지식을 습득하고 정리하는 능력 | 학습, 정보 탐색, 이해 및 정리 | 독서, 강의 시청, 공부 루틴, 키워드 노트 정리, 학습 복습 등 |

## 4. attributes.xlsx 테이블

attributes.xlsx 테이블은 다음의 시트로 구성된다.

- $definition 시트: 참고용 설명 시트
- attributes_attribute 시트: 정신/기술 능력치 항목 정보 정의
- attributes_level_mentality 시트: 정신 능력치의 Level 별 필요 경험치 밸런스
- attributes_level_skill 시트: 기술 능력치의 Level 별 필요 경험치 밸런스

### 4.1 $definition

| sheet_name | field_name | unique_index_group | data_format | null | enum_type | $desc |
| --- | --- | --- | --- | --- | --- | --- |
| attributes_config | field |  | int | FALSE |  | static data 이름 |
| attributes_config | value |  | int | FALSE |  | static data의 값 |
| attributes_attribute | attribute_id | 1 | int | FALSE |  | 능력치 id |
| attributes_attribute | type |  | string | FALSE | mentality, skill | 능력치 종류 |
| attributes_attribute | attribute_name |  | string | FALSE |  | 능력치 이름 |
| attributes_attribute | attribute_desc |  | string | TRUE |  | 능력치 설명문 |
| attributes_attribute | default |  | int | FALSE |  | 능력치 최초 값 |
| attributes_level_mentality | level | 1 | int | FALSE |  | 기준 레벨 |
| attributes_level_mentality | required_xp |  | int | FALSE |  | 기준 레벨에서 다음 레벨로 레벨업하기 위해 필요한 경험치 |
| attributes_level_skill | level | 1 | int | FALSE |  | 기준 레벨 |
| attributes_level_skill | required_xp |  | int | FALSE |  | 기준 레벨에서 다음 레벨로 레벨업하기 위해 필요한 경험치 |

**1) sheet_name**

- 시트 명

**2) field_name**

- 컬럼 명

**3) unique_index_group**

- 고유 값을 지정하기 위한 그룹 넘버
- 입력된 그룹 내에서 해당 필드에 동일한 값이 존재하면 안됨(에러 처리)

**4) data_format**

- 해당 필드의 data 형식
- int, string, boolean

**5) null**

- TRUE인 경우 해당 필드에서 null 값을 허용한다.
- FALSE인 경우 해당 필드에서 null 값이 존재하면 안된다. (에러 처리)

**6) enum_type**

- 해당 필드에서 사용하는 enum_type을 정의한다.

**7) $desc**

- 해당 필드 설명

## 5. 능력치 변경

각 능력치는 독립된 xp 값을 가진다.

### 5.1 성장

퀘스트를 수행하면 해당 퀘스트에 매핑된 능력치의 경험치(xp)를 획득한다.

경험치가 일정 수치에 도달하면, 능력치 수치(Level)가 1 상승한다.

xp는 정수 단위를 사용하며, attributes.xlsx 테이블의 attributes_level 시트를 참조한다.

경험치를 획득했을 때 레벨업을 한 경우 잔여 경험치는 이전되어야 한다.

- e.g. Level 1 (0xp)인 상태에서 +10xp를 획득한 경우 / level 1(8xp)가 되면 level 2가 됨
    - 결과: level 2 (2xp)

획득 경험치에 따라 다중 레벨업이 가능하다.

- e.g. Level 1 (0xp)인 상태에서 +30xp를 획득한 경우 / level 1(8xp)가 되면 level 2가 됨 & level 2(10xp)가 되면 level 3이 됨
    - 결과: Level 3 (12xp)

### 5.2 감소

현실에서 오랜 기간 해당 능력을 발휘하지 않을 경우 능력이 일부 감소한다.

반대로, 한번 체득한 능력이기 때문에 다시 해당 능력을 사용할 경우 과거의 지점에 빠르게 복귀할 수 있다.

이에 맞춰 '성장 정체'와 '성장 복구' 상태를 구현한다.

**[성장 정체]**

성장 정체는 유저가 관리하던 능력치를 오랜 기간 동안 성장시키지 않을 때 부여되는 능력치의 상태이다.

성장 정체를 부여하는 규칙은 다음과 같다.

- 현재 유저의 능력치 중 가장 높은 n개(attributes_config 시트의 stagnation_attribute_num 참조)의 능력치가
n일간(attributes_config 시트의 stagnation_attribute_days) 아무런 경험치 획득을 하지 않은 경우 해당 능력치에 '성장 정체' 상태를 부여한다.
    - 일 수는 마지막으로 경험치를 획득한 날짜부터 카운트한다.
    - 당일 23:59까지를 하루로 취급한다.
        - e.g. 성장 정체 상태가 아닌 케이스
            - 1) 7/1 14:00에 '체력' +10 xp 발생
            - 2) 7/7 23:59에 '체력' +5 xp 발생
        - e.g. 성장 정체 상태인 케이스
            - 1) 7/1 14:00에 '체력' +10 xp 발생
            - 2) 7/8 00:00가 되었는데 '체력'의 경험치 획득 0
            - 3) 7/8 00:00부로 '체력'은 성장 정체 상태가 됨

    성장 정체 상태가 되면 다음의 페널티가 적용된다.

    - 성장 정체 상태인 능력치는 이후 특정 일수마다 -n(attributes_config 시트의 stagnation_reduction_value_[n])레벨씩 감소한다.

    | 성장 정체 이후 n일 후 | 감소량 | (임시) 감소량 |
    | --- | --- | --- |
    | 당일 |  stagnation_reduction_value_1 | 1 |
    | 3일 후 |  stagnation_reduction_value_2 | 1 |
    | 7일 후 |  stagnation_reduction_value_3 | 2 |
    | 15일 후 |  stagnation_reduction_value_4 | 2 |
    | 30일 후 |  stagnation_reduction_value_5 | 2 |
    - 성장 정체로 인해 능력치 감소가 일어난 경우, 감소된 능력치의 경험치는 0으로 시작한다.
        - e.g. level 6 (20 xp)에서 능력치 감소로 - 1 level이 된 경우 ⇒ level 5 (0xp)
    - 성장 정체로 인한 능력치 감소로 인해 능력치가  attributes_attribute 시트의 default 값 미만으로 떨어지지 않도록 한다.
        - e.g. level 6에서 능력치 감소가 발생하더라도 default인 5 미만으로 떨어지지 않음

성장 정체 상태는 해당 능력치의 경험치를 얻게 된 순간 해제된다.

**[성장 복구]**

성장 복구 상태는 성장 정체 상태로 인해 능력치 감소가 발생한 경우 원래 능력치 레벨로 복구될 때까지 퀘스트 완료 시 추가 보너스 경험치를 획득하는 버프 상태이다.

성장 복구 상태가 적용되는 조건은 다음을 모두 만족해야 한다.

- 성장 정체 상태로 인해 능력치 레벨 감소가 발생했다.
- 감소되기 전 원본 레벨에 아직 도달하지 못했다.
- 해당 능력치의 경험치를 획득했다.

성장 복구 상태일 때 퀘스트를 완료하면, 해당 능력치의 경험치 보상을 획득할 때 config 시트의 growth_recovery만큼의 경험치를 추가로 획득한다.

- 자세한 경험치 획득 공식은 **[퀘스트 설정 시스템]**에서 추후 정한다.
    - e.g 퀘스트 경험치 산정 공식 예시: {(기본값 * [퀘스트 난이도] + [연속 수행 보너스]) * [동일 퀘스트 반복 페널티]} * [성장 복구 보너스]

### 5.3 수정/초기화

유저가 능력치를 직접 수정(edit)하거나 초기화하는 기능은 제공하지 않는다.

- 단, 추후 수정/초기화 아이템을 운영할 수 있음
